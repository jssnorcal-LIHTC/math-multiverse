# 5th Grade Math Games

## Files in this folder

- **HARDLINE-Fraction-Rider.html** — main game (open with double-click)
- **sur-ronster.webm** — required video asset (must stay in same folder)
- **sur-ronster.jpg** — poster/fallback image (must stay in same folder)
- **5th-Grade-Math-Practice-v1-26-0428.html** — printable mixed-practice worksheet with answer key

## How to play HARDLINE Fraction Rider

Open `HARDLINE-Fraction-Rider.html` in any modern browser.  Pick a difficulty (Rookie / Pro / Hardline), hit Drop In, solve fraction problems before three wrong answers wipe out Sur Ronster.
